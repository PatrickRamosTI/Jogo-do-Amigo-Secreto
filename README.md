# Amigo Secreto

Projeto simples em **HTML e JavaScript** para realizar sorteios de **Amigo Secreto** de forma prática e interativa.

---

## Funcionalidades
- Adicionar nomes a uma lista de participantes.
- Exibir os amigos adicionados em tempo real.
- Sortear um participante aleatório como "Amigo Secreto".
- Interface simples e intuitiva.

---

## Como executar o projeto

### 1. Clonar o repositório

git clone https://github.com/PatrickRamosTI/Jogo-do-Amigo-Secreto.git

### 2. Abrir o projeto

Basta abrir o arquivo `index.html` no navegador.
https://jogo-do-amigo-secreto-beryl.vercel.app/
